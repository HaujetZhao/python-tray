# Python Tray

一个 Python 模块，导入后，为脚本添加托盘图标，可隐藏控制台窗口。

## 功能特性

- **最小化到托盘**：窗口最小化时自动隐藏到托盘，而不是显示在任务栏
- **禁用关闭按钮**：防止误点击关闭按钮导致程序退出，可右键托盘菜单退出

## 使用方法

### 基础用法

```python
import tray

# 你的程序代码...
while True:
    # 程序主循环
    pass
```

就这么简单！导入模块后，托盘功能会自动启动。


## 依赖项

- `pystray` - 系统托盘图标库
- `Pillow` - 图像处理库

安装依赖：
```bash
pip install pystray Pillow
```

## 系统要求

- Windows 操作系统
- Python 3.6+

## 注意事项

- 仅在控制台窗口环境下生效（`.py` 文件）
- 在 IDE 内运行或使用 `.pyw` 文件时会自动跳过
- 程序运行时关闭按钮会被禁用，需通过托盘菜单退出

## 托盘菜单

- **显示/隐藏**：切换窗口可见性
- **退出程序**：安全退出并恢复关闭按钮

## 工作原理

1. 检测控制台窗口句柄
2. 禁用窗口关闭按钮，防止误关闭
3. 创建系统托盘图标
4. 监控窗口最小化事件，自动隐藏到托盘
5. 提供显示/隐藏切换和退出功能
